# Workplace Software and Skills

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/openstax/osbooks-workplace-software-skills)

_Workplace Software and Skills_ is a textbook published by [OpenStax](https://openstax.org/), a non profit organization that is part of [Rice University](https://www.rice.edu/).

The book can be viewed [online](https://openstax.org/details/books/workplace-software-skills), where you can also see a list of contributors.
